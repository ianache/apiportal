# Prompts for UI Design with Google Stitch

## Landing Pages

Quiero diseñar una aplicacion web con Vue para un portal de gestión de API que me permita registrar APIs de tipo Rest, realizar el diseño de sus endpoints, diseñar flujos de integración (inflow, outfloe, exception flow) qué expongan externamente las APIs Restull, para developers que ingresen al portal qué puedan suscribirse a las APIs o Integraciones publicadas. Debe ser un diseño limpio, moderno usando estilo de diseño UI de Apple

## Busquedas de Integraciones

Necesito añadir un diseño que permita en el menú integrations mostrar un listado de a modo de cards similares a las APIs de todas las integraciones. Se debe poder aplicar filtros por etiquetas, dominio (selector combobox) o input text. Los filtros deben aparecer en una barra en la parte superior t los resultados, páginados debajo de la barra de los filtros. Debe haber 2 modos de visualización de resultados: mostrar cards (default) o listado en forma tabular.
