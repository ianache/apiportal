# Feature 05: Consulta de APIs

# Requerimientos

Necesito que desde el menu alteral opcion "Explorer" se acceda a una vista donde usuario con cualquier role puedan realizar consultas de las APIs siguiendo varios criterios: por dominio, etiqueta, nombre del API y poder filtrar segun estado de actual de las APIs (según los estados definidos se puede seleccionar cero o varias). El resultado se debe mostrar en forma de tabla o card mostrando la información general del servicio (tomar como base el Card actual usado en "APIs Catalog" donde se visualiza cada card.
